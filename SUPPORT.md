# Support

NowFlow Community support happens through public project channels:

- Use GitHub issues for reproducible bugs and scoped feature requests.
- Use discussions or community channels for questions and implementation help.
- Include versions, setup steps, logs, and screenshots when they help reproduce
  a problem.

For enterprise support, managed deployment, governance, hosted UI generation,
agent operations, or private implementation work, visit https://nowflow.io.

Never share real secrets, customer data, production URLs, staging URLs, or
database dumps in public support requests.
