# Security Policy

## Supported Project

This repository contains NowFlow Community, the open-source edition. Security
fixes are accepted for the current default branch and current tagged community
releases.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability.

Report security issues by emailing security@nowflow.io with:

- A short description of the issue
- Steps to reproduce
- Impact and affected versions, if known
- Any logs or proof-of-concept details that do not expose real secrets

We will acknowledge reports as quickly as we can and coordinate disclosure once
a fix is available.

## Secret Handling

Do not include real credentials, API keys, tokens, private keys, production URLs,
staging URLs, database dumps, or customer data in issues, pull requests, logs, or
example files.
